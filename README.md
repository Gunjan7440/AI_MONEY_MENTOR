# AI Money Mentor

💰 **AI Money Mentor** is a web-based application that helps users track their income, expenses, and savings, provides AI-generated financial advice, visualizes finances with charts, and calculates the FIRE (Financial Independence, Retire Early) goal.

---

## **Features**

- ✅ Login system for user access  
- ✅ Enter monthly income, expenses, and savings  
- ✅ Calculate net worth, suggested investments, and emergency fund  
- ✅ AI chatbot to answer questions about saving, investing, tax, FIRE, and loans  
- ✅ Visual charts using Chart.js for financial overview  
- ✅ History of previous financial entries  
- ✅ FIRE goal calculation based on expenses  

---

## **How to Use**

1. Open `login.html` in a web browser  
2. Enter any username and password, then click **Login**  
3. On the main page (`index.html`):  
   - Enter **Income**, **Expenses**, and **Savings**  
   - Click **Analyze Financial Health** to see:  
     - Net Worth  
     - Suggested Investments  
     - Emergency Fund  
     - Health Score  
     - FIRE Goal  
4. Chat with the AI chatbot to get financial advice:  
   - Ask about saving, investing, tax, FIRE, loans, stocks, budget, etc.  
5. View your history or clear it anytime  

---

## **Technologies Used**

- HTML, CSS, JavaScript  
- Chart.js for interactive financial charts  
- localStorage for storing user data and history
